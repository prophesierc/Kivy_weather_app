## Getting Started
</br>

You must get an API Key at [https://openweathermap.org/](https://openweathermap.org/)

### Prerequisites

1. Install requirements 
   ```sh
   $ pip install -r requirements.txt
   ```
2. Enter your API in `myapi.py`
   ```py
   api_key = 'OPENWEATHERMAP API KEY'
   ```
